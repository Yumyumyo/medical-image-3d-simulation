# Medical Image 3D Interaction and Cutting Simulation

## Introduction
This project focuses on developing an interactive 3D system for medical image visualization and cutting simulation.  
The system allows users to explore 3D medical structures and perform simulated cutting operations for better understanding and potential surgical planning.

## Features
- 3D medical image visualization
- Interactive rotation and navigation
- Cutting simulation
- Real-time rendering and feedback

## Technologies
- Python
- 3D visualization techniques
- Basic image processing
